# IDAM STUDIO — Architecture Portfolio Website

**이담건축 공식 포트폴리오 사이트 리포지토리**

Next.js 기반의 미니멀하고 정갈한 아키텍처 스튜디오 웹사이트입니다. 브랜드의 ‘본질로 완성하는 건축’ 철학을 온라인 경험으로 번역하여 소개, 작업 사례, 상담/문의, 위치 안내 등을 간결하게 제공합니다.
